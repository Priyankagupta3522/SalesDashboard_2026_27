# SalesDashboard_2026_27

## 📊 Project Overview
This repository contains an **Interactive Power BI Dashboard** showcasing **sample sales data for 2026-27**.  
The project demonstrates skills in **data visualization, KPI tracking, and business insights** using Microsoft Power BI.  
⚠️ *Note: The dataset used is demo/sample data, not actual business data.*

---

## 🎯 Key Features
- **KPIs**: Total Sales, Profit, Quantity, Orders, Segments  
- **Regional Insights**: Sales distribution across U.S. regions (Central, East, South, West)  
- **Category Analysis**: Quantity and sales by product categories (Binders, Furnishings, Paper, Storage, etc.)  
- **Segment Analysis**: Consumer, Corporate, and Home Office sales breakdown  
- **Interactive Visuals**: Map, bar charts, pie charts, and KPI cards  

---

## 🛠️ Tools & Technologies
- **Microsoft Power BI** – for dashboard creation and visualization  
- **Sample Sales Dataset** – demo dataset for practice and portfolio purposes  
- **Data Modeling** – relationships between Customer, Product, Region, and Sales data  

---

## 📌 Insights Highlighted
- **2M Total Sales** achieved in 2026-27  
- **286K Total Profit** across all categories  
- **Top Segment**: Home Office with 1.16M sales  
- **Regional Comparison**: Central, East, South, and West performance breakdown
